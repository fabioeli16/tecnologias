-TIPOS DE PROYECTO REACT

1-React “a mano” con CDN (sin build tools)..................................


2-React con npm + Babel + Webpack (configuración manual)...................


3-Create React App (CRA) (casi en desuso)..................................

Características
	*Antes era el estándar
	*Todo configurado automáticamente
	*Webpack oculto

Problemas
	*Lento
	*Pesado
	*Ya no es recomendado oficialmente
#COMANDOS:
npx create-react-app my-app
cd my-app
npm start


4-Vite + React (RECOMENDADO)...............................................

Características
	Arranque ultra rápido
	Hot Reload instantáneo
	Configuración mínima

#COMANDOS:
npm create vite@latest my-app
cd my-app
npm install
npm run dev

5-Next.js (React con backend incluido).....................................
Características
	SSR (renderizado del lado servidor)
	Routing automático
	Backend integrado (API routes)
	Excelente SEO

#COMANDOS:
npx create-next-app@latest
